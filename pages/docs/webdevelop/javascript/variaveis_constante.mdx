# Variáveis
Como declarar uma variável em JavaScript?
#### Sem valor:
```javascript
var user;
console.log(user)
```

#### Declarar variável e atribuir valor:
```javascript
var email = 'email@email.com';
console.log(email)
```

#### Substituição de valor da variável:
```javascript
email = 'joao@email.com';
console.log(email)
```
Observe que não é necessário declarar novamente a variável email, apenas atribuir um novo valor
